# 🛒 E-Commerce Web Application (MERN Stack)

A full-stack e-commerce web application built using the MERN stack. The app allows users to browse products, add items to cart, place orders, and managing their accounts, while providing an admin dashboard for managing products, orders and users. This repo is the front-end part.

## ✨ Features

### 👤 User Features

* User authentication (Login / Register / Logout)
* Browse products
* View product details
* Add / remove / change quantity items from cart
* Place orders
* Review orders
* Manage account

### 🛠️ Admin Features

* Add, update, and delete products
* Manage users
* Manage orders

## 🧰 Tech Stack

### Frontend

* React.js
* Zustand (state management)
* React Router
* Axios
* CSS Modules

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/omarAlaa/E-Commerce-frontend.git
cd E-Commerce-frontend
```

---

### 2️⃣ Setup Frontend

```bash
cd frontend
npm install
npm start
```

---

### 3️⃣ Setup Backend

```bash
cd backend
npm install
```

Create a `.env` file in the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Run the backend:

```bash
npm run dev
```

---

## 🔐 Environment Variables

Make sure to add your environment variables and **never expose them publicly**.

Example:

```env
MONGO_URI=...
JWT_SECRET=...
```
